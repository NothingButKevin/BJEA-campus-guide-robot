## Installation Steps

1.	Clone the repository

    ```bash
    git clone https://github.com/NothingButKevin/BJEA-campus-guide-robot
    cd BJEA-campus-guide-robot
    ```

2. Install dependencies

    ```bash
    pip install -r requirements.txt
    ```

3. Run the project

    ```bash
    python main.py
    ```
