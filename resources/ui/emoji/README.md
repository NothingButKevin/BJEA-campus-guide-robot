# Emoji PNG 资源

此目录存放各机器人状态对应的大号 emoji PNG 图片。

## 所需文件

| 文件名 | 对应状态 | 参考 emoji |
|--------|---------|-----------|
| idle.png | IDLE | 😊 |
| listening.png | LISTENING | 🎧 |
| matching.png | MATCHING | 🤔 |
| confirming.png | CONFIRMING | ❓ |
| chatting.png | CHATTING | 💬 |
| navigating.png | NAVIGATING | 🚗 |
| arrived.png | ARRIVED | 🎉 |
| shutdown.png | SHUTDOWN | 😴 |

## 规格

- 尺寸：512x512（或更大）
- 格式：PNG，透明背景
- 获取方式：从 [Twemoji](https://github.com/twitter/twemoji) 下载 SVG 并转 PNG

## 回退

若 PNG 文件缺失，GUI 将自动使用大号 Unicode 文字字符作为回退显示。
